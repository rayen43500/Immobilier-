# Immobilier - Home Page

Modern, responsive home page for real-estate listings with a hero banner, advanced filters, property cards, and call-to-action.

## Requirements
- Node.js 18+

## Install
```bash
npm install
```

## Dev
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Deploy to GitHub Pages
Repository: https://github.com/rayen43500/Immobilier-.git

```bash
npm install
npm run deploy
```

After deploy, enable GitHub Pages in the repo settings and select the `gh-pages` branch.
